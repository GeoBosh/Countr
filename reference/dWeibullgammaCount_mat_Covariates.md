# Univariate Weibull Count Probability with gamma and covariate heterogeneity

Univariate Weibull Count Probability with gamma and covariate
heterogeneity

## Usage

``` r
dWeibullgammaCount_mat_Covariates(
  x,
  cc,
  r,
  alpha,
  Xcovar,
  beta,
  t = 1,
  logFlag = FALSE,
  jmax = 100L
)
```
