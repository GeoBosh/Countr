# Univariate Weibull Count Probability with gamma heterogeneity

Univariate Weibull Count Probability with gamma heterogeneity

## Usage

``` r
dWeibullgammaCount_mat(
  x,
  shape,
  r,
  alpha,
  time = 1,
  logFlag = FALSE,
  jmax = 100L
)
```

## Arguments

- r, alpha:

  double gamma parameters
