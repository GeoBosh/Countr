# Create a formula for renewalCount

Create a formula for renewalCount

## Usage

``` r
CountrFormula(response, ...)
```

## Arguments

- response:

  the formula for the "main" parameter. It also specifies the response
  variable.

- ...:

  additional arguments for the ancilliary parameters.

## Value

a Formula object suitable for argument `formula` of
[`renewalCount()`](https://geobosh.github.io/Countr/reference/renewalCount.md).
